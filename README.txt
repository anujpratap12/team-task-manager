TEAM TASK MANAGER

Project Overview:
A full-stack Team Task Manager web application where users can create projects, manage tasks, update task status, and track progress with authentication and protected routes.

Tech Stack:
Frontend:
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- bcryptjs

Deployment:
- Frontend: Vercel
- Backend: Railway

Features:
- User Signup/Login
- JWT Authentication
- Protected Routes
- Project Creation
- Task Creation
- Task Status Update
- Dynamic Dashboard
- Navbar Navigation
- Logout Functionality
- MongoDB Database Integration
